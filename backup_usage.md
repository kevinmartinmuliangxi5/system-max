# Python文件备份脚本使用说明

## 功能特性

- ✅ 自动备份指定目录下的所有 `.py` 文件
- ✅ 在 `backup` 文件夹中创建带时间戳的备份
- ✅ 递归遍历子目录
- ✅ 保持原有目录结构
- ✅ 完善的异常处理
- ✅ 详细的日志记录
- ✅ 自动生成备份摘要
- ✅ 支持命令行参数

## 使用方法

### 基本用法

```bash
# 备份当前目录下的所有.py文件
python backup_py_files.py
```

### 高级用法

```bash
# 备份指定目录
python backup_py_files.py -s /path/to/source

# 自定义备份文件夹名称
python backup_py_files.py -s . -b my_backup

# 查看帮助信息
python backup_py_files.py -h
```

## 命令行参数

| 参数 | 简写 | 说明 | 默认值 |
|------|------|------|--------|
| --source | -s | 源目录路径 | . (当前目录) |
| --backup | -b | 备份目录名称 | backup |

## 输出说明

### 备份目录结构

```
backup/
└── backup_20260203_143025/
    ├── backup_summary.txt    # 备份摘要文件
    ├── brain.py
    ├── dealer.py
    ├── setup.py
    └── ...                   # 保持原有目录结构
```

### 日志文件

脚本会在当前目录生成日志文件：`backup_20260203_143025.log`

### 备份摘要

自动生成的 `backup_summary.txt` 包含：
- 备份时间
- 源目录和备份目录
- 备份文件数量
- 失败文件数量
- 总文件大小

## 退出码

| 退出码 | 说明 |
|--------|------|
| 0 | 备份完全成功 |
| 1 | 部分文件备份失败 |
| 2 | 未找到任何Python文件 |
| 130 | 用户中断操作 |

## 注意事项

1. backup 目录中的文件不会被备份（避免重复备份）
2. 备份时会保持原有的目录结构
3. 使用 `copy2` 保留文件的元数据（修改时间等）
4. 支持中文路径和文件名
