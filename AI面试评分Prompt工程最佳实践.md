## 1. 评分溯源Prompt设计

### 1.1 核心原则

评分溯源是指AI给出的每个评分都必须能够追溯到考生回答的具体内容，避免主观臆断。

```
┌─────────────────────────────────────────────────────────────┐
│                  评分溯源三大原则                          │
├─────────────────────────────────────────────────────────────┤
│ 1. 引用原话: 必须使用考生回答中的具体语句作为依据        │
│ 2. 说明原因: 解释为什么这句话加/扣分                       │
│ 3. 关联维度: 明确指出影响哪个评分维度                     │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 评分溯源Prompt模板

#### 基础模板

### 1.3 评分溯源Prompt要点总结

| 要点 | 说明 | 示例 |
|------|------|------|
| 明确引用格式 | 使用特殊符号标记引用 | > 考生原话：「...」 |
| 关联价值观 | 每个评分都要关联岗位核心价值观 | 符合了执法岗位的"依法行政"原则 |
| 量化影响 | 说明具体加/扣多少分 | 因此在原则性维度加5分 |
| 避免空泛 | 不使用模糊词汇 | 避免"表现良好"，要说"体现了..." |

## 2. 引用原话的Prompt模式

### 2.1 引用原话的重要性

引用原话是评分溯源的核心，它确保：
1. **可信度**：考生可以验证评分依据
2. **公平性**：基于实际表现而非主观印象
3. **可追溯**：每个评分都有明确来源
4. **教学价值**：让考生明白哪里好、哪里不好

### 2.2 引用原话Prompt技巧

#### 技巧1：明确引用符号

```python
QUOTE_SYMBOL_PROMPT = """
【引用原话的符号规范】

在评分报告中引用考生原话时，必须遵守以下符号规范：

1. 使用「」符号包裹引用内容
   示例：
   > 考生原话：「执法过程中要始终坚持依法行政」

2. 长段引用使用省略号截断
   示例：
   > 考生原话：「执法过程中要始终坚持依法行政...这样才能实现执法效果和社会效果的统一」

3. 多句连续引用使用分号分隔
   示例：
   > 考生原话：「首先要依法行政；其次要程序规范；最后要文明执法」

4. 引用后必须紧跟分析
   格式：
   > 考生原话：「...」
   >
   > 点评：[分析内容]

【禁止行为】
❌ 使用模糊的"考生提到"而不引用具体语句
❌ 引用后不进行点评分析
❌ 修改或意译考生原话
"""
```

#### 技巧2：精准定位语句

```python
PRECISE_QUOTE_PROMPT = """
【精准定位原话技巧】

在引用考生原话时，请遵循以下技巧：

1. **最小引用原则**
   只引用体现评分点的最小完整语意单元
   ✅ 好：「执法要坚持依法行政」（5个字，明确）
   ❌ 差：「关于这个问题，我认为执法过程中首先要明确，执法的目的是为了维护城市秩序，所以在执法时要坚持依法行政」（太长，重点不突出）

2. **保留关键修饰词**
   引用时保留体现态度和程度的修饰词
   ✅ 好：「始终坚持依法行政」（"始终"体现态度）
   ❌ 差：「坚持依法行政」（失去了态度的体现）

3. **完整语义单元**
   引用必须保持语义完整，不能断章取义
   ✅ 好：「虽然情况紧急，但也要坚持程序正义」（完整表达）
   ❌ 差：「但也要坚持程序正义」（缺少前提条件）

4. **位置标记法**
   对于长回答，可以使用"开头/中间/结尾"标记引用位置
   示例：
   > 考生原话（回答开头）：「首先，我认为...」
   > 考生原话（回答结尾）：「...才能实现执法效果和社会效果的统一」
"""
```

#### 技巧3：避免断章取义

```python
CONTEXT_AWARE_QUOTE_PROMPT = """
【避免断章取义的引用原则】

在引用考生原话时，必须注意：

1. **保留上下文**
   如果考生的观点有前提条件，引用时必须包含

   考生说：「在原则性问题上，我们要坚持底线；但在非原则性问题上，可以灵活处理」

   ✅ 正确引用（完整）：
   > 考生原话：「在原则性问题上，我们要坚持底线；但在非原则性问题上，可以灵活处理」
   > 点评：这体现了考生既能坚持原则，又懂得灵活变通

   ❌ 错误引用（断章取义）：
   > 考生原话：「可以灵活处理」
   > 点评：这会让考生显得没有原则

2. **引用转折句的完整结构**
   转折句必须完整引用，体现"虽然...但是..."

   ✅ 正确：「虽然要严格执法，但也要注重人性化管理」
   ❌ 错误：「要注重人性化管理」

3. **引用限制性条件**
   有条件的观点必须引用条件

   ✅ 正确：「对于初犯或情节轻微的情况，可以先教育劝导」
   ❌ 错误：「可以先教育劝导」（缺少前提条件）

【强制要求】
- 引用前检查：去掉这句话的某个部分，是否改变原意？
- 如果会改变，必须保留完整内容
- 宁可引用长一点，也不要误解考生意思
"""
```

## 3. 岗位画像融入Prompt

### 3.1 岗位画像的作用

岗位画像确保评分具有针对性和个性化，避免"一刀切"的通用评分。

```
┌────────────────────────────────────────────────────────────┐
│              岗位画像三层次结构                        │
├────────────────────────────────────────────────────────────┤
│ 第一层：岗位大类（决定了基础权重）                      │
│   🛡️ 行政执法类 | 🤝 窗口服务类 | 📝 综合管理类    │
├────────────────────────────────────────────────────────────┤
│ 第二层：具体岗位（决定了核心价值观）                     │
│   城管执法队员 | 政务大厅窗口人员 | 机关文秘          │
├────────────────────────────────────────────────────────────┤
│ 第三层：滑块调节（决定了权重微调）                      │
│   服务柔性 0-100% | 原则刚性 0-100% | ...           │
└────────────────────────────────────────────────────────────┘
```

    scoring_guidance=scoring_guidance,
    user_answer=user_answer
