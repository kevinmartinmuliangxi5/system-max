## 🎓 使用示例

### 完整工作流

```bash
# 1. 环境验证
python .ralph/scripts/check_dependencies.py
python .ralph/scripts/integration_test.py

# 2. Brain规划任务
python brain_v3.py "实现用户认证系统"

# 生成:
# - .janus/project_state.json (蓝图)
# - .ralph/specs/实现用户认证系统.md (规格)
# - .ralph/diagrams/实现用户认证系统.xml (流程图)

# 3. Dealer生成指令
python dealer_v3.py

# 生成完整指令:
# - 任务概览
# - 文件内容
# - Context Engineering上下文
# - 双记忆经验
# - Superpowers规则
# - 质量门控清单

# 4. Worker执行（粘贴指令给Claude Code）
# Worker会:
# - 完整实现所有功能
# - 编写完整测试
# - 系统化质量自检
# - 输出learning标签
# - 输出完成信号

# 5. 并行处理多个任务（可选）
python .ralph/tools/parallel_brain.py \
  "任务1" "任务2" "任务3" \
  --mode tmux
```

### 关键命令

```bash
# 依赖检查

# 并行Brain
```
